## Install

```bash
$ sudo apt-get install vim vim-gnome ack-grep ctags build-essential cmake python-dev
$ cd /tmp
$ wget https://raw.github.com/weekface/vimfile/master/install.sh
$ chmod +x install.sh
$ ./install.sh
```
