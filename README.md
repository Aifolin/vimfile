## Ubuntu Install

```bash
$ curl -s https://raw.githubusercontent.com/weekface/vimfile/master/install | bash
$ ~/.vim/bundle/YouCompleteMe/install.py --gocode-completer # if you want golang env.
$ vim +GoInstallBinaries +qall # if you want golang env.
```
